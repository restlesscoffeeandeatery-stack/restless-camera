📲 Install di Android

1
Tap tombol "Install Aplikasi" di bawah
2
Pilih "Install" saat popup muncul
3
Icon Absensi Restless akan muncul di homescreen
4
Buka app → izinkan kamera → absen seperti biasa

📲 Install di iPhone/iPad

1
Tap ikon Share (□↑) di bawah layar Safari
2
Scroll ke bawah, pilih "Add to Home Screen"
3
Tap "Add" di pojok kanan atas
🌐 Buka Langsung di Browser
